# PrimeTrade Internship Project

## Overview
This is a Django-based backend project for task management with user authentication and role-based access. Admins can manage all tasks, while regular users can manage only their own tasks.

## Features
- User registration and login
- Role-based access control (Admin / User)
- CRUD operations for tasks
- Token-based authentication (JWT)
- Simple REST API endpoints
